Methods

bool brep_vertices_topology_is_valid() const

Checks if the brep has topologically valid unique vertices, i.e. has vertices which verify: Each unique_vertex can only be associated to one corner. Each corner can only be internal to one object (surface or block). Each corner is a boundary of every line it is associated to.

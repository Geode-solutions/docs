SolidSwapFacetInfo swap_facet(const PolyhedronFacet& facet)

Swap a facet. Two tetrahedra should be incident to this facet. These two tetrahedra are set as inactive and three new tetrahedra are created.

Parameters:

facet

Index of the PolyhedronFacet to swap. @warn Old tetrahedra are set as inactive but not deleted.

 
SolidSwapEdgeInfo swap_edge(const PolyhedronFacetEdge& edge, index_t apex)

Swap an edge. The edge can be on border. The tetrahedra around given edge are set as inactive and new tetrahedra are created.

Parameters:

edge

Index of the PolyhedronFacetEdge to swap.

apex

Index of the vertex used as the new facets fan common vertex. @warn Old tetrahedra are set as inactive but not deleted.

 
SolidRemoveMultipleAdjacencyInfo remove_double_adjacency(const PolyhedronFacetEdge& edge)

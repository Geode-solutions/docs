template class geode::PolygonalSurfaceView
==========================================

.. toctree::
	:hidden:

A mesh view of a :ref:`PolygonalSurface <doxid-classgeode_1_1_polygonal_surface>` meaning a subset of a :ref:`PolygonalSurface <doxid-classgeode_1_1_polygonal_surface>` without copying any geometry or microtopology, it only "views" the selected subeset and acts like it is a complete mesh with all the API from vertices to polygons, attributes included. Modification on the :ref:`PolygonalSurface <doxid-classgeode_1_1_polygonal_surface>` will not be reproduced on

the corresponding view.

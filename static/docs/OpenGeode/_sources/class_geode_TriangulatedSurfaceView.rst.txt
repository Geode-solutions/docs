template class geode::TriangulatedSurfaceView
=============================================

.. toctree::
	:hidden:

A mesh view of a :ref:`TriangulatedSurface <doxid-classgeode_1_1_triangulated_surface>` meaning a subset of a :ref:`TriangulatedSurface <doxid-classgeode_1_1_triangulated_surface>` without copying any geometry or microtopology, it only "views" the selected subeset and acts like it is a complete mesh with all the API from vertices to triangles, attributes included. Modification on the :ref:`TriangulatedSurface <doxid-classgeode_1_1_triangulated_surface>` will not be reproduced on

the corresponding view.

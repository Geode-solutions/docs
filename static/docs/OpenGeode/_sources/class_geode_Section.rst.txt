A :ref:`Section <doxid-classgeode_1_1_section>` is a 2D :ref:`GeoRepresentation <doxid-classgeode_1_1_geo_representation>` composed of :ref:`Corners <doxid-classgeode_1_1_corners>`, :ref:`Lines <doxid-classgeode_1_1_lines>`, and :ref:`Surfaces <doxid-classgeode_1_1_surfaces>`. This class provides classes for range-based iteration on :ref:`Component <doxid-classgeode_1_1_component>` boundaries and incidences.

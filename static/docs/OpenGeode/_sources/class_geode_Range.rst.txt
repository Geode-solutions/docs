class geode::Range
==================

.. toctree::
	:hidden:

This class can be used to iterate over integer loop. Example: = C++98 loop = for( index_t i = 0; i < n; i++ ) { // do something }

.. code-block:: none

	= C++11-like loop =

for( auto i : Range( n ) ) { // do something }

A Boundary Representation is a :ref:`GeoRepresentation <doxid-classgeode_1_1_geo_representation>` composed of :ref:`Corners <doxid-classgeode_1_1_corners>`, :ref:`Lines <doxid-classgeode_1_1_lines>`, :ref:`Surfaces <doxid-classgeode_1_1_surfaces>` and :ref:`Blocks <doxid-classgeode_1_1_blocks>`. This class provides classes for range-based iteration on :ref:`Component <doxid-classgeode_1_1_component>` boundaries and incidences.
